### nauka_mnozenia
Program w Python, z przeznaczeniem dla dzieci z klas 1-3, prosty, szybki dla Linux i Windows

## 26-11-2023
Wprowadziłem delikatne poprawki do programu, teraz można podać dolną i górną granicę mnożenia liczb przez siebie

## 27-11-2023
Aktualizacja wersji z 1.0 do 2.0
Wprowadziłem pewne zmiany w programie, program ma zmienione statystyki i możliwość zapisywania i wyświetlania na ekranie błędnych odpowiedzi. Przydatna funkcja która pozwoli określić braki w nauce dziecka.
